# New-PnPTenantTemplate

## SYNOPSIS
Creates a new tenant template object

## SYNTAX 

```powershell
New-PnPTenantTemplate [-Author <String>]
                      [-Description <String>]
                      [-DisplayName <String>]
                      [-Generator <String>]
```

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
$template = New-PnPTenantTemplate
```

Creates a new instance of a tenant template object.
