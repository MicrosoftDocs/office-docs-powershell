# Get-PnPFolderItem

## SYNOPSIS
List content in folder

## SYNTAX 

```powershell
Get-PnPFolderItem [-ItemType <String>]
                  [-ItemName <String>]
                  [-FolderSiteRelativeUrl <String>]
                  [-Web <WebPipeBind>]
                  [-Connection <SPOnlineConnection>]
```
