# Add-PnPTenantTheme

## SYNOPSIS
Adds or updates a theme to the tenant.

## SYNTAX 

```powershell
Add-PnPTenantTheme -Identity <ThemePipeBind>
                   -Palette <ThemePalettePipeBind>
                   -IsInverted <Boolean>
                   [-Connection <SPOnlineConnection>]
```

## DESCRIPTION
Adds or updates atheme to the tenant.
