# New-PnPTermGroup

## SYNOPSIS
Creates a taxonomy term group

## SYNTAX 

```powershell
New-PnPTermGroup -Name <String>
                 [-Id <Guid>]
                 [-Description <String>]
                 [-TermStore <Id, Name or Object>]
                 [-Connection <SPOnlineConnection>]
```

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
New-PnPTermGroup -GroupName "Countries"
```

Creates a new taxonomy term group named "Countries"
