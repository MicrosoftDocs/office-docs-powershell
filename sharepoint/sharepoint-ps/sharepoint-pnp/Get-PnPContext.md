# Get-PnPContext

## SYNOPSIS
Returns the current context

## DESCRIPTION
Returns a Client Side Object Model context

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
$ctx = Get-PnPContext
```

This will put the current context in the $ctx variable.

## OUTPUTS

### Microsoft.SharePoint.Client.ClientContext
