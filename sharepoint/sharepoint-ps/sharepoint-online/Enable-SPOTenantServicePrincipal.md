# Enable-SPOTenantServicePrincipal

## SYNOPSIS
Enables the current tenant's "SharePoint Online Client" service principal.

> [!WARNING]
> This is a preview feature and should not be used in production.

## SYNTAX

```powershell
Enable-SPOTenantServicePrincipal
```

## DESCRIPTION
Enables the current tenant's "SharePoint Online Client" service principal.
