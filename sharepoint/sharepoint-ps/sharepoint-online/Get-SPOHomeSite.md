# Get-SPOHomeSite

## SYNOPSIS

Returns the home site url for your tenant.

## SYNTAX

```powershell
Get-SPOHomeSite [<CommonParameters>]
```

## DESCRIPTION

Use this cmdlet to get the home site url for your tenant.

## EXAMPLES

### Example 1

```powershell
Get-SPOHomeSite
```

This example returns the home site url for your tenant.
