# Get-CsMainlineAttendantSupportedVoices

## SYNOPSIS

The Get-CsMainlineAttendantSupportedVoices cmdlet returns a list of languages that are supported for use with Mainline Attendant.

## SYNTAX

```
Get-CsMainlineAttendantSupportedVoices
```

## DESCRIPTION

The Get-CsMainlineAttendantSupportedVoices cmdlet returns a list of languages that are supported for use with Mainline Attendant.

## EXAMPLES

```
Get-CsMainlineAttendantSupportedVoices
```
